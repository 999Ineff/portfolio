#01 Excel / Data
Auto-Format Report Macro

A macro that formats your weekly/monthly report with consistent styling, headers, and formatting — one click instead of 15 minutes of manual work.

The Prompt
Create an Excel VBA macro that formats my [report type] report. It should: - Apply [company/preferred] colors to headers - Set column widths appropriately for the data - Add borders to the data range - Format numbers as [currency/percentage/number] - Freeze the top row - Add a "Last Updated" timestamp in cell [A1/other] The data typically has [X] columns and starts in row [2/other].
Info to Provide
  • Your column headers (e.g., "Date, Customer, Amount, Status")
  • Preferred colors (hex codes or "blue headers, alternating row colors")
  • Number formats needed (currency, percentages, dates)
  • Any special formatting rules (negative numbers in red, etc.)
Pro Tip

Ask AI to also add a keyboard shortcut (like Ctrl+Shift+F) so you can run it instantly.

How to Deploy
  1. Press Alt+F11 in Excel to open VBA editor
  2. Insert → Module
  3. Paste the code
  4. Save as .xlsm (macro-enabled)
  5. Run via View → Macros or your keyboard shortcut
#02 Excel / Data
Data Cleanup Macro

Cleans messy data: removes duplicates, trims whitespace, fixes formatting inconsistencies, standardizes text case.

The Prompt
Create an Excel VBA macro that cleans data in my spreadsheet: - Remove duplicate rows based on [column(s)] - Trim extra spaces from all text cells - Convert [column] to [UPPERCASE/lowercase/Proper Case] - Standardize [phone numbers/dates/other] format - Remove blank rows - Sort by [column] Show a message box when complete with count of rows cleaned.
Info to Provide
  • Which columns contain what type of data
  • What "clean" looks like for your data (date format, phone format)
  • Which column determines duplicates
  • Any values that should be flagged or highlighted
Pro Tip

Ask AI to create a backup of your data to a new sheet before cleaning, so you can undo if needed.

#03 Excel / Reports
One-Click Dashboard Refresh

Refreshes all pivot tables, data connections, and recalculates formulas — ready for your morning meeting.

The Prompt
Create an Excel VBA macro that: - Refreshes all pivot tables in the workbook - Refreshes all data connections/queries - Recalculates all formulas - Updates the "Last Refreshed" cell to current date/time - Shows a completion message Make it handle errors gracefully if any refresh fails.
Info to Provide
  • What external data sources you connect to (if any)
  • Where you want the timestamp displayed
  • If any sheets should be excluded from refresh
Pro Tip

Ask AI to add a button to your dashboard sheet that runs this macro — visual and easy.

#04 File Management
Downloads Folder Organizer

Python script that sorts your downloads by file type into folders, renames with dates, and cleans up old files.

The Prompt
Create a Python script that organizes my Downloads folder: - Move PDFs to a "Documents" subfolder - Move images (jpg, png, gif) to "Images" subfolder - Move videos to "Videos" subfolder - Move installers (exe, msi) to "Installers" subfolder - Rename files to include today's date: originalname_2024-01-15.ext - Delete files older than [30/60/90] days (optional, ask first) Create the subfolders if they don't exist. Also create a .cmd file so I can double-click to run it.
Info to Provide
  • Your Downloads folder path (usually C:\Users\YourName\Downloads)
  • Any custom categories you want (e.g., "Spreadsheets" for xlsx)
  • Files or folders to ignore
  • Whether to actually delete old files or just list them
How to Deploy
  1. Save the Python file as organize_downloads.py
  2. Save the .cmd file in the same folder
  3. Double-click the .cmd to run
  4. Optional: Add to Windows Task Scheduler to run daily
#05 Email / Communication
Email Template Generator

Python script that reads customer data from Excel and generates personalized emails ready to send.

The Prompt
Create a Python script that: - Reads customer data from an Excel file (columns: [Name, Email, Company, etc.]) - Generates personalized emails using this template: """ Hi [First Name], [Your email body with placeholders] Best regards, [Your name] """ - Saves each email as a .txt file named "email_[CustomerName].txt" - Creates a summary of all emails generated Include a .cmd file to run it easily.
Info to Provide
  • Your Excel column names exactly as they appear
  • Your email template with placeholders marked
  • Any conditional content (different message for different customer types)
  • Where to save the output files
Pro Tip

Ask AI to add the ability to send emails directly through Outlook — but start with generating text files to review first.

#06 Data Transfer
System-to-System Format Converter

Takes an export from System A and reformats it for import into System B — eliminating manual copy-paste.

The Prompt
Create a Python script that converts data between two formats: INPUT (from [System A]): - File type: [CSV/Excel/other] - Columns: [list them] OUTPUT (for [System B]): - File type: [CSV/Excel/other] - Columns needed: [list them] - Column mappings: - [SystemA_Column] → [SystemB_Column] - [etc.] Also apply these transformations: - [Date format change, text cleanup, etc.] Save output with timestamp in filename.
Info to Provide
  • Sample export file from System A (or describe exact format)
  • Import requirements for System B (or sample template)
  • Any data transformations needed (date formats, value mappings)
  • Columns to ignore or add
#07 Text Processing
Clipboard Text Cleaner

Hotkey script that cleans whatever's in your clipboard — removes formatting, fixes spacing, standardizes text.

The Prompt
Create a Python script that: - Reads text from clipboard - Cleans it by: - Removing extra whitespace and line breaks - Converting smart quotes to regular quotes - Removing hidden characters - [Any other cleaning you need] - Puts cleaned text back in clipboard - Shows a small notification "Clipboard cleaned!" Make it run with a keyboard shortcut using the pynput library. Hotkey: [Ctrl+Shift+C or other]
Info to Provide
  • What "clean" means for your use case
  • Your preferred keyboard shortcut
  • Any text transformations (UPPERCASE, remove emails, etc.)
Pro Tip

Ask AI to make it run on Windows startup so the hotkey is always available.

#08 Excel / Data Entry
Data Entry Form Macro

Creates a popup form for data entry that validates input and adds to your spreadsheet consistently.

The Prompt
Create an Excel VBA UserForm for data entry with these fields: - [Field 1]: [text/dropdown/date] - [Field 2]: [text/dropdown/date] - [Field 3]: [text/dropdown/date] Validation rules: - [Field] is required - [Field] must be a valid [email/number/date] When submitted: - Add data to next empty row in [Sheet name] - Auto-fill today's date in [column] - Clear form for next entry - Keep form open for multiple entries
Info to Provide
  • All fields you need to capture
  • Which fields are required vs optional
  • Dropdown options for any list fields
  • Which sheet and columns receive the data
#09 Reporting
PDF Report Generator

Python script that pulls data from Excel and generates a formatted PDF report.

The Prompt
Create a Python script that generates a PDF report: - Read data from [Excel file] - Create a PDF with: - Title: [Report Name] - Date generated - Summary statistics (total, average, etc.) - Data table with the key columns - [Any charts if needed] Use the reportlab library (or fpdf). Save as "Report_YYYY-MM-DD.pdf" Include a .cmd file to run easily.
Info to Provide
  • Source Excel file location and sheet name
  • What summary stats you want (totals, averages, counts)
  • Which columns to include in the report
  • Any grouping or filtering needed
#10 Backup / Safety
Auto-Backup Script

Backs up important files/folders to a backup location with dated folders.

The Prompt
Create a Python script that backs up my important files: Source folders to backup: - [C:\path\to\folder1] - [C:\path\to\folder2] Backup destination: [D:\Backups or network path] The script should: - Create a dated folder (Backup_2024-01-15) - Copy all files from source folders - Skip files that haven't changed (compare dates) - Log what was backed up - Delete backups older than [30] days - Show summary when complete Include a .cmd file and instructions to schedule it weekly.
Info to Provide
  • Exact paths of folders to backup
  • Where backups should go (local drive, external, network)
  • How many days of backups to keep
  • Any file types to exclude (e.g., .tmp files)
Scheduling (Windows)
  1. Open Task Scheduler
  2. Create Basic Task
  3. Set trigger (weekly, daily, etc.)
  4. Action: Start a program → your .cmd file
  5. Done — runs automatically!
REFERENCE

Quick Reference: What to Tell AI

The Better Your Description, The Better Your Tool

Instead of... Say this...
"Make it work with my files" "Files are in C:\Reports, named report_MMYYYY.xlsx"
"Format the dates" "Convert dates from MM/DD/YYYY to YYYY-MM-DD"
"Clean the data" "Remove duplicates based on email column, trim whitespace, make names Title Case"
"Add error handling" "If file not found, show message and exit. If data is empty, create blank report."
